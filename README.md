**FTN HUD Manager** es una plataforma de escritorio avanzada diseñada para elevar la calidad de producción en transmisiones de Counter-Strike 2. Centraliza el control estético y el análisis de datos en una sola interfaz robusta y optimizada.

---

## 🌎 Language / Idioma

### **English**
**FTN HUD Manager** is a next-generation production tool that bridges the gap between live game data and professional broadcast overlays. Built for competitive environments, it offers zero-latency synchronization and deep data integration.

* **Dynamic HUD Control**: Modify colors, layouts, and branding elements in real-time without restarting the game or interrupting the stream.
* **Pro Analytics Dashboard**: Comprehensive tracking for matches, players, and teams with a clean, intuitive UI.
* **Performance Monitoring**: Dedicated tools for coaches to analyze round-by-round efficiency and player performance.
* **Hardware-Locked Security**: Custom licensing system that links accounts to specific hardware (HWID) for maximum security.

### **Español**
**FTN HUD Manager** es una herramienta de producción de nueva generación que conecta los datos del juego en vivo con overlays profesionales. Diseñada para entornos competitivos, ofrece sincronización de latencia cero e integración profunda de datos.

* **Control Dinámico de HUD**: Modificá colores, diseños y elementos de marca en tiempo real sin reiniciar el juego ni interrumpir el stream.
* **Dashboard de Analíticas Pro**: Seguimiento integral de partidas, jugadores y equipos con una interfaz limpia e intuitiva.
* **Monitoreo de Rendimiento**: Herramientas dedicadas para que los coaches analicen la eficiencia ronda por ronda y el desempeño de los jugadores.
* **Seguridad por Hardware**: Sistema de licencias personalizado que vincula cuentas a hardware específico (HWID) para garantizar la máxima seguridad.

---

## 🛠️ Tech Stack / Tecnologías
* **Desktop Layer**: Electron (Custom Main Process).
* **Backend**: Node.js & Express con arquitectura de WebSockets para sincronización instantánea.
* **Frontend**: React.js con Tailwind CSS para una interfaz de alta respuesta.
* **Database & Auth**: Supabase para gestión de usuarios y estados en tiempo real.
* **Game Link**: Integración avanzada de GSI para CS2.
